Git palvelimella:

Mikäli valitsit tämän, harkitse yhä uudelleen Githubin käyttöä.

Git repon perustaminen palvelimella

TKTL opiskelijat menkää palvelimelle users.cs.helsinki.fi ssh-yhteydellä
Tällöin kansio jossa olet on /home/<username>/ voit tarkistaa kansion sijaintisi komennolla pwd

Nyt luo kansio, josta git reposi tulee. mkdir reponNimi.git ja siirry luomaasi kansioon cd reponNimi.git/
Tässä kansiossa komenna git init --bare Näin olet luonut git repositorion.

Git repon kloonaaminen

Mene koneellasi kansioon, johon haluat git kansion syntyvän. Ja komenna git clone username@palvelin.fi:/polku kansioosi/repo.git ja TKTL tapauksessa: git clone username@users.cs.helsinki.fi:/home/username/reponNimi.git

nyt kansioon jossa olit syntyi uusi kansio reponNimi. Kopioi tähän kansioon ne tiedostot jotka haluat sinne git:in halintaan.
